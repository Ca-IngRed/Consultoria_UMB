# ColSoft Solutions - Consultora de Software

## Propósito de la página

Esta página web fue desarrollada como una landing page informativa para una consultora de software.

El objetivo principal es presentar de forma organizada los servicios de la empresa, información general sobre la consultora y ofrecer un medio de contacto para posibles clientes interesados en soluciones tecnológicas.

Entre los servicios que presta la empresa estan servicios de desarrollo web, análisis de datos, automatización y consultoría tecnológica.

---

## Aplicación de semántica HTML5

Para mejorar la organización se utilizaron las siguientes etiquetas:

- `<header>` para el encabezado principal.
- `<nav>` para el menú de navegación.
- `<main>` para el contenido principal.
- `<section>` para dividir las secciones de la pagina.
- `<article>` para representar los servicios ofrecidos por la empresa por la empresa.
- `<aside>` información adicional.
- `<footer>` para el pie de página.

La funcion de estas etiquetas es ayudar a mejorar la estructura del documento y facilitan la comprensión para navegadores y lectores de pantalla fundamental para las opciones de accesibilidad.

---

## Accesibilidad aplicada

La página incluye diferentes prácticas de accesibilidad:

- Uso de atributos `alt` en todas las imágenes.
- Asociación correcta entre `<label>` e inputs del formulario.
- Uso de `aria-label` en la navegación y botones importantes.
- Diseño responsive para dispositivos móviles.
- Contraste adecuado entre colores de fondo y texto.
- Indicadores visuales de enfoque (`focus`) para navegación con teclado al presiona la tecla TAB.

---

## Instrucciones para visualizar la página

1. Descargar o clonar el proyecto.
2. Abrir la carpeta del proyecto en Visual Studio Code.
3. Abrir el archivo `index.html` en un navegador web.

Opcionalmente, se recomienda utilizar la extensión **Live Server** de Visual Studio Code para visualizar la página. También es importante mantener la estructura original de carpetas para que las imágenes y archivos CSS funcionen correctamente.
